A general Food App built with:

1. Node JS
2. Express
3. MySql
4. Sequelize
5. Joi validator

After cloning don't forget to run:

[npm i]

Note:
Kindly create a .env file in the current directory and save the following variables in it:
DB_DIALECT = mysql
DB_NAME = food_app
DB_HOST = localhost
DB_USER = [YOUR_USERNAME]
DB_PASSWORD = [YOUR_PASSWORD]
PORT = 3000
HOST = 'localhost'
